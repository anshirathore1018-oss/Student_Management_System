# 🎓 Student Management System (Python)

A **Student Management System** built using **Python OOP** and **Pandas** with complete **CRUD operations**.

---

## 🚀 Features

✅ Add Student
✅ Search Student
✅ Update Student
✅ Delete Student
✅ Show All Students in Table Format
✅ Duplicate Student ID Validation
✅ CSV Data Storage

---

## 🛠️ Technologies Used

* **Python**
* **Pandas**
* **Object-Oriented Programming (OOP)**
